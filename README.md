# Venera 中文/常用漫画源整合包

包含两类：

1. 本地随包 JS：拷贝漫画、Komiic、包子漫画、爱看漫（波洞）、MangaDex、comick、少年ジャンプ＋。
2. 远程官方 JS：GoDa漫画（G站漫画）、漫画1234、漫蛙吧、再漫画、漫画柜、漫画人、漫小肆、嗨皮漫画、优酷漫画。

说明：COCO、vomic、次元漫画、古风漫画、酷看漫画、酷漫屋、来漫画、六漫画、漫百库、漫神、七夕漫画、速漫库等没有在当前可用 Venera 配置包中找到现成 JS，所以没有强行加入。强行加入只会出现空白、404 或图片加载失败。

上传 GitHub 时，把本目录全部文件放到仓库根目录。Venera 导入：
